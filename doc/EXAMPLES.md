## Полезные команды

### Сборка проекта
```bash
make              # Собрать всё
make server       # Только сервер
make client       # Только клиент
make clean        # Очистить
```

### Запуск
```bash
./server          # Запуск сервера
./client          # Запуск клиента
make run-server   # Запуск через make
make run-client   # Запуск через make
```

### Отладка
```bash
# Запуск с отладочной информацией
gdb ./server
gdb ./client

# Проверка утечек памяти
valgrind --leak-check=full ./server
valgrind --leak-check=full ./client
```

### Остановка всех процессов
```bash
pkill -f "./server"
pkill -f "./client"
```

## Советы по игре

1. **Начинайте со стратегии:**
   - Первая попытка: используйте разные цифры (например, 0123)
   - Анализируйте количество быков и коров
   - Исключайте неподходящие варианты

2. **Используйте логику:**
   - Если получили 0 быков и 0 коров - все 4 цифры неправильные
   - Если 4 коровы - все цифры правильные, но все не на месте
   - Если 1 бык и 3 коровы - одна на месте, остальные переставить

3. **Минимизируйте попытки:**
   - Опытные игроки могут угадать за 4-6 попыток
   - Рекорд: 3 попытки (с большой долей удачи)

## Расширенные сценарии

### Турнир между игроками
Создайте несколько игр и соревнуйтесь, кто быстрее угадает:

```bash
# 3 игрока, 3 терминала
Игрок 1: создает "tournament1"
Игрок 2: создает "tournament2"
Игрок 3: создает "tournament3"

Каждый пытается угадать своё число быстрее других!
```

### Командная игра
Несколько игроков в одной игре помогают друг другу:

```bash
# Игра на 4 игроков
Игрок 1: пробует 0123
Игрок 2: пробует 4567
Игрок 3: пробует 8901
Игрок 4: комбинирует информацию и угадывает!
```
